## 一个适用于AstrBot的自动呼叫群则批的插件。

主要用于群组内的玩家认证、管理，以及在检测到（关键词或IP地址）时自动呼叫列表内的所有玩家。

### 功能特性

- 群组隔离：不同群聊的玩家列表互不干扰。

- 玩家认证：通过简单的指令将群友认证为“则批”，并记录特定昵称/机签。

- 列表管理：支持查看当前群所有认证玩家，支持移除认证。


### 全网呼叫：

- 监听群消息，检测到关键词 "ymyz" 时自动触发。

- 检测到 IP:端口（ipv4）格式（如 8.134.214.80:36203）时自动触发。

- 触发时发送“全网呼叫则则人”并 @列表内所有玩家。


### 安装方法

1.在 AstrBot 的插件目录中执行：

```
git clone https://github.com/waterladleh/astrbot_plugin_ymyz.git
```

2.在 AstrBot 管理面板中重启 AstrBot。

### 数据存储

玩家数据保存在插件目录下的

```
 zepi_list.json
```

数据结构支持群组隔离，格式为 JSON。

### 指令列表

| 指令      | 格式                 | 说明                          |
| --------- | -------------------- | ----------------------------- |
| /你是则批 | /你是则批 @某人 昵称 | 将被 @ 的用户加入本群列表     |
| /开除则籍 | /开除则籍 @某人      | 将被 @ 的用户从本群列表中移除 |
| /则批列表 | /则批列表            | 查看本群所有则批              |
